\section{Methodology}
%% ============================================================================

\subsection{Dataset and Embedding}

We use the Sachdeva et al.\ dataset of 10,826 moral dilemmas from Reddit's ``Am I the Asshole'' forum. Each dilemma includes a top human comment alongside rationales from seven LLMs (GPT-3.5, GPT-4, Claude, Bison, Gemma, Mistral, Llama), each providing 2--3 rationales per dilemma for 216,497 LLM rationales total.

All rationales are encoded with Kaleido-XL, an encoder-decoder model trained on ValuePrism (31K moral situations with value annotations). Mean-pooling the encoder's final hidden states produces a 2048-dimensional embedding per rationale. Because Kaleido's representation space is structured around moral knowledge, dimensionality in this space is a meaningful proxy for moral reasoning diversity. To control for encoder-specific bias, we replicate all core analyses with Sentence-BERT (\texttt{all-mpnet-base-v2}, 768-dim), which has no moral-specific training.

\subsection{Analysis Pipeline}

We measure dimensionality with PCA, computing two metrics per source: the participation ratio $\text{PR} = (\sum \lambda_i)^2 / \sum \lambda_i^2$, which gives the effective number of variance-carrying dimensions, and comp90, the number of components needed for 90\% cumulative variance. Comp90 is more sensitive to the eigenvalue tail. We include matched-$n$ subsampling (5 seeds), dilemma-matched subsampling, word-count-quintile binning, and kernel PCA (RBF) as robustness checks.

We then test for asymmetry by projecting each source's data onto the other's principal components and comparing the variance captured in each direction. The residuals from this cross-projection (the parts of each source's representation that the other's PCs cannot capture) are qualified using Kaleido's decoder, which generates a moral value label per rationale. We collapse the 324 unique labels into semantic clusters (Sentence-BERT + $k$-means) and measure how value diversity changes in sliding windows ($w{=}200$, step${=}50$) along the reconstruction error axis.

Last, we measure local LLM density around each human rationale using $k$-nearest neighbors ($k{=}50$) in PCA-reduced space, to determine whether the gap reflects regions LLMs avoid entirely or regions they populate less densely.

\section{Results}
%% ============================================================================

\subsection{Dimensionality Gap}

Human rationales need 448 PCA components to reach 90\% variance. Individual LLMs need 329--341, and the pooled all-LLM set needs 347 (Table~\ref{tab:summary}). The cumulative variance curves (Figure~\ref{fig:cumvar}) show that human variance accumulates more slowly, with a longer tail past the point where LLM curves flatten.

When we subsample each LLM to $n{=}10{,}826$ to match the human count, the gap is 35--41\%: LLMs need only 317--332 components. Dilemma-matched subsampling (selecting exactly 1 of each LLM's 2--3 rationales per dilemma) shifts comp90 by only 2--5 components. The gap persists across all word-count quintiles (30--47\% per bin), and kernel PCA yields near-identical PR rankings ($r = 0.984$), ruling out nonlinear structure.

\subsection{Asymmetric Overlap}

The gap is directional. Projecting all-LLM data onto the human PCA basis ($k{=}448$) captures 87.3\% of LLM variance. In the other direction, projecting human data onto all-LLM PCs ($k{=}347$), only 81.6\% is captured, a 5.7pp asymmetry (Figure~\ref{fig:heatmap}). LLM reasoning sits largely within the human subspace, but human reasoning extends into directions that LLM PCs do not span. Per-model, LLM PCs capture 77.5--81.3\% of human variance while human PCs capture 85.8--87.7\% of each LLM. In the SBERT replication the asymmetry widens to 9.4pp.

\subsection{Moral Content in the Extra Dimensions}

The extra human dimensions could reflect writing style (Reddit slang, profanity, informal register) rather than moral content. To test this, we use Kaleido's decoder to generate a moral value label for each rationale, sort all 10,826 human rationales by their reconstruction error under LLM PCs, and measure value diversity in sliding windows. The correlation between reconstruction error and unique value clusters per bin is $r = 0.897$ ($p < 0.0001$, Figure~\ref{fig:gradient}): the harder a rationale is for LLM PCs to capture, the more diverse the moral values it invokes. The reverse direction (LLM rationales under human PCs) yields $r = 0.823$. TF-IDF analysis on the surface text does identify stylistic markers among the high-residual rationales, but the gradient is measured through decoded value labels and is independent of register.

\subsection{Frequency Compression}

KNN analysis shows that LLMs are not absent from the high-residual regions of embedding space, but sparser there. The mean distance from each human rationale to its 50 nearest LLM neighbors correlates with reconstruction error at $r = 0.969$ (binned) and $r = 0.846$ (per-rationale); the density ratio (LLM distance / human distance) reaches $r = 0.988$ at the bin level. LLMs cover most of the space but concentrate their mass in a smaller region, populating the periphery far less than humans do.

%% ============================================================================
\section{Analysis}
%% ============================================================================

The results are consistent across the four stages: human moral reasoning occupies a higher-dimensional subspace, the gap is asymmetric, the extra dimensions carry moral content, and the mechanism is frequency compression rather than categorical absence.

We tested four confounds. The gap survives matched-$n$ subsampling with less than 1 component of seed variation against a 114--131 component gap. It persists within every word-count quintile. Kernel PCA produces near-identical rankings ($r = 0.984$), ruling out nonlinear structure. And Sentence-BERT, which has no exposure to GPT-4 moral training data, replicates the gap at proportionally similar magnitude; the cross-projection asymmetry is larger in SBERT (9.4\% vs.\ 5.7\%), not smaller.

The value diversity gradient ($r = 0.897$) connects the extra dimensions to moral content: rationales that LLM PCs fail to reconstruct are the ones that invoke the most diverse moral values. The KNN density results show this is a continuous gradient ($r = 0.969$ binned) rather than a hard boundary: LLMs under-populate the periphery of the value distribution, but they do not avoid it entirely.

There are limitations. The analysis relies on a single dataset (Reddit AITA dilemmas), and Kaleido's decoder was trained on GPT-4-generated value labels, which introduces circularity when qualifying residuals involving GPT-4 as a source. The SBERT replication controls for encoder bias but not this decoder dependency. The seven LLMs in the dataset represent a particular generation of models, and newer systems may behave differently.

\section{Plan for Additional Analysis}
%% ============================================================================

We plan to stratify dilemmas by community consensus level (low $<0.5$, medium $0.5$--$0.8$, high $>0.8$ agreement) and test whether the dimensionality gap depends on moral controversy. If LLMs compress diversity as much on clear-cut dilemmas as on contested ones, the compression is structural rather than context-dependent. Bootstrap resampling at matched $n$ will control for unequal bucket sizes.

We also plan to test whether categorical analysis is appropriate for this data. Prior work (e.g., Russo et al.) treats moral values as discrete categories and measures diversity by counting them. We will run $k$-means on the rationale embeddings and evaluate cluster quality via silhouette scores and gap statistics. If scores are near zero, the value space is better understood as a continuum, which would support our geometric approach.

All seven LLMs fall in a narrow comp90 range (329--341), but their residual profiles may differ. Per-model cross-projection residuals mapped to specific value categories could show whether the compression is a shared training artifact or varies across models.

Last, if the compression is a frequency bias rather than a capability limitation, it may be partially recoverable through prompting. We plan to test persona-based prompts, explicit diversity instructions, and chain-of-thought with moral framework enumeration, measuring whether any of these shift the LLM embedding distribution toward higher dimensionality.
